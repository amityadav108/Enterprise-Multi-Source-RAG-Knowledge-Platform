
# Enterprise Multi-Source RAG Knowledge Platform

## Overview

A production-style Retrieval-Augmented Generation (RAG) system capable of answering questions from multiple enterprise data sources.

Supported Sources:
- PDFs
- CSV files

Tech Stack:
- Python
- FastAPI
- FAISS
- Scikit-learn
- Pandas

Features:
- Multi-source document ingestion
- Chunking pipeline
- Vector search using FAISS
- Citation-based responses
- Conversation memory
- REST APIs with FastAPI
- Swagger documentation
